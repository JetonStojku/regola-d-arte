# Regola D'arte image rules

- Use real clinic images only when already approved and web-optimized.
- Use labeled placeholders when the intended image is known but the final asset is not selected.
- Every placeholder must describe the exact image to replace it.
- Keep image and placeholder dimensions stable with aspect ratios or fixed responsive heights.
- Do not publish before/after clinical photos unless consent, pairing, and final copy are explicitly confirmed.
- Do not invent treatment outcomes, reviews, ratings, credentials, prices, or patient stories.
- Write meaningful alt text for informative images and keep decorative images silent.
- Never overwrite original files in `materials/photos/`.
