# Regola Photo Sections

Before editing photos:

1. Read `.codex/rules/regola-images.md`.
2. Decide whether the section needs an approved real image or a placeholder.
3. Use placeholders when final image selection, consent, or treatment copy is not confirmed.
4. Keep placeholders descriptive: image type, replacement intent, and stable dimensions.
5. Mirror photo slots and labels across Italian and English pages.

Before finishing:

- Confirm no unapproved before/after clinical image was published.
- Confirm placeholders are clear enough for the owner to replace later.
- Confirm meaningful real images have alt text.
- Run `npm run build` in `website/` when code changed.
