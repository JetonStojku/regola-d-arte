# Photo placeholder system

## Goal

Use reusable photo placeholders wherever the site needs more visuals but final images are not confirmed.

## Required rules / skills

- Read `.codex/rules/regola-images.md`.
- Use `.codex/skills/regola-photo-sections/SKILL.md`.

## Implementation plan

1. Use reusable placeholder components instead of blank space or generic cards.
2. Label each placeholder with the intended image type.
3. Describe what image should replace it.
4. Preserve dimensions with aspect ratio classes to prevent layout shift.
5. Use real images only when approved and optimized.

## Acceptance criteria

- [ ] Placeholder purpose is clear to the site owner.
- [ ] Placeholders are visually polished and responsive.
- [ ] Real archived before/after images are not published by default.

## Verification

Inspect all placeholders on home, services, service detail, about, and contact pages.
