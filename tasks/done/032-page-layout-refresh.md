# Page layout refresh

## Goal

Refresh home, about, services, service detail, and contact pages so information is balanced and supported by visuals.

## Required rules / skills

- Read `.codex/rules/regola-layout.md`.
- Read `.codex/rules/regola-images.md`.
- Use `.codex/skills/regola-layout-review/SKILL.md`.
- Use `.codex/skills/regola-photo-sections/SKILL.md`.

## Implementation plan

1. Replace repeated heavy split surfaces with balanced content bands.
2. Avoid large empty columns next to dense stacked cards.
3. Add real images where already approved and placeholders where images are pending.
4. Keep service and contact information scannable in evenly weighted grids.
5. Mirror changes in Italian and English.

## Acceptance criteria

- [ ] No page has a sparse left column paired with an overloaded right column.
- [ ] Each major page includes meaningful visual content or image placeholders.
- [ ] Italian and English layouts remain equivalent.

## Verification

Build the site and visually check `/it/`, `/en/`, `/it/chi-siamo/`, `/it/servizi/`, service pages, and contact pages.
