# Before / after section

## Goal

Add and maintain a before/after work section using placeholders until clinical photos are approved.

## Required rules / skills

- Read `.codex/rules/regola-images.md`.
- Use `.codex/skills/regola-photo-sections/SKILL.md`.

## Implementation plan

1. Keep the section anchor as `#before-after`.
2. Use paired before and after placeholder slots with stable aspect ratios.
3. Include treatment description guidance without inventing outcomes.
4. Do not publish archived before/after images unless consent and final copy are confirmed.

## Acceptance criteria

- [ ] Before/after placeholders are visible and easy to replace.
- [ ] The section does not imply unverified treatment results.
- [ ] Italian and English copy is present.
